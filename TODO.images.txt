# Static Images Folder Structure

Create the following folder and image files for the project:

```
static/
└── images/
    ├── youtube-logo.png       # YouTube logo  
    ├── tiktok-logo.png        # TikTok logo
    ├── instagram-logo.png     # Instagram logo
    ├── x-logo.png             # X (Twitter) logo
    ├── facebook-logo.png      # Facebook logo
    ├── telegram-logo.png      # Telegram logo
    ├── patreon-logo.png       # Patreon logo
    ├── paypal-logo.png        # PayPal logo
    ├── buymeacoffee-logo.png  # Buy Me a Coffee logo
    ├── kofi-logo.png          # Ko-fi logo
    ├── email-logo.png         # Email logo
    ├── instagram-mini-logo.png # Instagram mini logo for contact section
    ├── facebook-mini-logo.png  # Facebook mini logo for contact section
    ├── x-mini-logo.png         # X (Twitter) mini logo for contact section
    └── patreon-mini-logo.png   # Patreon mini logo for contact section
```

You should download appropriate SVG or PNG logos for each service from their brand assets pages or use a reputable icon library. Make sure all logos have a transparent background and consistent size (recommended 200x200px for main logos and 100x100px for mini logos).
